If you configured your MSA to not consume EU identities, the matching datasets you receive from the GOV.UK Verify Hub will be formatted according to the legacy JSON matching schema:

{

        "cycle3Dataset": {
            "properties": {

            "type": "string"

        "matchingDataset": {
            "properties": {
                "addresses": {
                    "items": {
                        "properties": {

                                "items": {
                                    "type": "string"
